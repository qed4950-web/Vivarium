# 📸 스마트 가드닝 AIoT

## **데이터 수집 가이드라인 (Data Collection Guideline)**

Roboflow 식물 건강·병충해·성장 모델 구축을 위한 이미지 수집 규칙

---

# 🎯 1. 데이터 수집 목적

이 데이터는 **식물 상태를 AI가 자동으로 분석**할 수 있도록 만드는 핵심 자원이다.
모델이 인식해야 할 주요 대상:

1. **식물 종류(Plant Species)**
2. **식물 건강 상태(Growth & Health)**
3. **병충해(Disease & Pests)**
4. **스트레스(Stress: 수분 부족/과습/광량 부족 상태)**

---

# 📁 2. 데이터셋 구조 권장

프로젝트를 4개로 나누는 것을 강력히 추천:

```
1. plant_species/      → 식물 종류 분류
2. plant_health/       → 건강/성장 추적
3. plant_disease/      → 병반/색 변화
4. plant_pests/        → 벌레 인식
```

대규모 혼합 프로젝트보다 **문제별 분리**가 더 정확하고 관리도 쉬움.

---

# 📷 3. 촬영 환경 기준

## ✔ 밝기(조도)

* **부드러운 자연광**이 가장 이상적
* 백색 LED 조명은 가능하나 그림자가 심하면 제외
* 너무 어둡거나 탁한 사진은 사용률 낮음

## ✔ 배경

* 식물 뒤 배경이 **깔끔**할수록 좋음
* 책상·벽·선반·커뮤니티 가드닝 배경 모두 OK
* 흙, 주변 화분, 난잡한 배경도 **실제 환경 반영용으로 일부 포함**

비율 권장:

* 깨끗한 배경 70%
* 실제 사용 환경(난잡) 30%

---

# 🪴 4. 식물 종류(Plant Species) 데이터 수집 규칙

수집 목표는 **가장 많이 쓰일 식물 기준**:

예시:

* 바질(Basil)
* 로즈마리(Rosemary)
* 민트(Mint)
* 토마토(Tomato)
* 고추/고추나무(Pepper)
* 허브류
* 몬스테라, 스투키 등 실내용 관엽식물

식물별로 최소 목표:

| 종류            | 최소 이미지 | 권장 이미지 |
| ------------- | ------ | ------ |
| 허브류(바질, 민트 등) | 30     | 80~150 |
| 관엽식물          | 30     | 60~100 |
| 채소/열매류        | 30     | 80~150 |

여기서 **각 식물은 “식물 전체 + 잎 클로즈업”을 모두 포함**해야 함.

---

# 🌿 5. 건강 상태(Growth & Health) 수집 규칙

AI는 잎의 **색, 크기, 개수, 형태**를 보고 성장 정도를 판단함.

그래서 아래 이미지를 모두 포함해야 함:

### ✔ 건강한 잎(good leaves)

* 진한 녹색
* 탄력 있음
* 결점 없음
* 신선한 광택

### ✔ 약간 스트레스 받는 잎

* 색이 조금 연함
* 끝이 마르고 갈변 시작
* 잎 가장자리가 말림
* 전체적으로 퇴색

### ✔ 심각한 스트레스

* 잎 전체 노랗게 변색
* 크게 처짐
* 잎 끝이 타들어감
* 줄기까지 힘없이 늘어짐

각 상태별로 **10장 이상** 확보 권장.

---

# 🍂 6. 병충해(Disease) 수집 규칙

병해 데이터는 모델 성능을 결정하는 가장 중요한 부분.

최소 포함해야 하는 케이스:

### ✔ 병반(Spot)

* 작은 점
* 노란 Spot
* 갈색 Spot
* 원형 또는 불규칙 형태

### ✔ 잎 변색(Discoloration)

* 크게 노랗게
* 큰 부분 갈변
* 얼룩 패턴

### ✔ 잎 손상(Damage)

* 구멍
* 찢김
* 모서리 떨어짐

### ✔ 곰팡이(Mold)

* 하얀 곰팡이
* 가루 형태

각 병해 유형은 최소 **10장**, 권장 **30장**.

---

# 🐛 7. 벌레(Pest) 수집 규칙

아파트/실내 환경에서 가장 많이 등장하는 벌레 기준:

* 진딧물(Aphid)
* 응애(Spider Mite)
* 혹파리(Fungus gnat)

벌레 데이터는 가능하면 **매크로 촬영** 권장.

---

# 🔢 8. 이미지 수량 가이드

최소 목표량:

| 분야       | 최소 수량 | 권장 수량   |
| -------- | ----- | ------- |
| 식물 종류 식별 | 100   | 300~600 |
| 건강 상태    | 100   | 200~400 |
| 병해/병반    | 60    | 150~250 |
| 벌레       | 20    | 50~100  |

최종적으로 500~1,500장 규모면 **상업용 MVP 수준** 성능 충분함.
